### How I work with AI

I run **AI agent teams** &mdash; a main orchestrator AI agent coordinates a roster of specialised AI agents, each in its own isolated workspace, all moving in parallel. AI reads, understands and analyses the codebase first; discusses the plan with me like a team member; builds a thorough plan before any code is written. Then it writes, tests, debugs and ships &mdash; opens a real browser, walks every acceptance criterion, captures screenshots, refuses to mark anything done until every check passes, and iteratively fixes itself until the feature reaches its goal. **For bugs**: AI reproduces the failure, traces it to the root cause, proposes a fix plan plus a prevention plan (tests, guardrails, refactors), then implements both. Every lesson learned, every quirk, every decision flows into a persistent AI memory so the next session already knows.
